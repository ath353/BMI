- Một chương trình tính BMI đơn giản áp dụng những kiến thức cơ bản của Python như: input(), ép kiểu dữ liệu, f - string, toán tử số học, ternary expression
- BMI (Body Mass Index) - Chỉ số khối cơ thể. Giúp đánh giá tình trạng cân nặng có phù hợp với chiều cao không.

- Công thức: BMI = cân nặng (kg) ÷ chiều cao² (m)
